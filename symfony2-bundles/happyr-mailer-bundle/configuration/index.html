Here is the configuration reference. The config is under

[code]
# config.yml
happy_r_mailer:
class: HappyR\MailerBundle\Services\MailerService
from:
name: webmaster
email: webmaster@example.com

[/code]

Mandatory parameters

There are no mandatory parameters to this bundle. But you really should change form.email and from.name.

Optional parameters

class

The class we should use as mailer service. You should never need to change this parameter.

from.email

The email address email will be sent from.

from.name

The name of the person that sends the email.
